# Overview

The **Cases** toolkit is where your organization records and tracks customer concerns. Each case captures the issue, the contact involved, related documents, a comment thread, and the workflow that drives it to resolution.

Use the pages in this section to:

* Create a new case — see [Creating Cases](creating-cases.md).
* Find, view history on, or update an existing case — see [Editing Cases](edit-cases/README.md).
* Attach documents to a case — see [Case Documents](case-documents.md).
* Add or edit comments on a case — see [Case Comments](case-comments/README.md).
