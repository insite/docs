# IDE

The following editors and tools are in active use on the Shift iQ codebase:

* **Visual Studio 2022** — primary IDE for the .NET back end.
* **Visual Studio Code** — preferred editor for the front-end TypeScript / React code and for editing documentation.
* **Insomnia** — REST/GraphQL client used to exercise the API while developing endpoints.
