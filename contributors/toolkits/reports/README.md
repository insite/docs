# Reports

The Reports toolkit is the read-side surface for platform data. It packages saved queries and presentation rules so administrators can answer operational questions without writing SQL.

This section covers:

* [Monitors](monitors.md) — long-running watches that surface anomalies and threshold breaches.
* [Queries](queries.md) — saved parameterized queries that back the reports.
* [Report types](report-types.md) — the catalog of report shapes available to administrators.
